# Find J&L Security services

J&L Security (established 2011; SSAIB, CHAS, FIA and BAFE accredited) installs and
maintains security and fire-safety systems for homes and businesses across Essex
and Greater London.

## How to use

- Read the structured overview at `https://jandlsecurity.co.uk/llms.txt` or the
  expanded version at `https://jandlsecurity.co.uk/llms-full.txt`.
- Or call the MCP server at `https://jandlsecurity.co.uk/mcp`:
  - `list_services` returns all services.
  - `get_service_details` with `{ "service": "<slug>" }` returns features and benefits.
- Any page also returns markdown when requested with `Accept: text/markdown`.

## Services

- **Burglar alarms** (`burglar-alarms`): wireless and wired intruder alarms, monitoring, app control, police-response eligible.
- **CCTV systems** (`cctv-systems`): 4K cameras, night vision, remote viewing, cloud storage.
- **Fire alarms** (`fire-alarms`): BS 5839-1 commercial and BS 5839-6 domestic/HMO systems; BAFE-certified maintenance.
- **Access control** (`access-control`): fobs, cards, biometric readers, video intercoms, maglocks.
- **Security lighting** (`security-lighting`): PIR and LED security and emergency lighting (BS 5266-1).
- **Fire risk assessments** (`fire-risk-assessments`): accredited assessments with in-house remedial works.
